# Welcome to your Student Vault!

This is an Obsidian vault template meant for college/university students. It's made to be easy to use out-of-the-box for all kinds of majors and is light on plugins to ensure that it loads quickly on your devices.

## What is Obsidian?

Obsidian is a lightweight Markdown file editor. All of your notes are stored locally as **.md** text files, making them easily readable, editable, and viewable while offline. It's also highly customizable. While this vault is geared towards students, there are all sorts of use cases for a versatile text editor like Obsidian.

To learn more about Markdown syntax, click [here](https://www.markdownguide.org/basic-syntax/). You'll need to learn Markdown to format your text. Don't worry, it's simple! (Or you can just use the editing toolbar, pictured below.)

One of the most integral features of Obsidian is its use of wikilinks. By surrounding text with double brackets `[[like this]]`, you can link notes to each other and click on them to navigate to other notes. 

## How do I get started?

First, you'll want to [download Obsidian](https://obsidian.md/download) for your device as well as the [Student Vault](https://github.com/emiyen/student-vault/archive/refs/heads/main.zip) itself. 

Unzip the Student Vault **.zip** file. Navigate into the **student-vault-main** folder and you should see a duplicate of it inside. Take that one out and place it on your computer somewhere accessible, such as your Documents folder.

Upon opening Obsidian for the first time, you'll see a window that looks like this:

<img src="https://github.com/emiyen/student-vault/blob/main/00%20SYSTEM/attachments/CleanShot%202026-07-07%20at%2016.14.47@2x_6558ac070973650e745c34399cb0b26bb6006ddeb18da01a37b692da49c1dc34.jpeg?raw=true">

Select **Open** next to the 'Open folder as vault' option. A file navigator will pop up. Navigate to the **student-vault-main** that you took out of the duplicate folder and select it.

A window will pop up prompting you to either enable the vault's plugins or browse the vault in Restricted Mode.

<img src="https://github.com/emiyen/student-vault/blob/main/00%20SYSTEM/attachments/CleanShot%202026-07-07%20at%2016.23.19@2x_52ecbdd570666f8a79907d194c9b3c180753070253eb2e9d1e2ba44ad305e307.jpeg?raw=true">

You can check what plugins are used by clicking [here](https://github.com/emiyen/student-vault#what-plugins-are-used-in-this-vault) before downloading the vault. Once you've looked them over and gauged their safety, you can click **Trust author and enable plugins**. You can also browse in Restricted Mode and view the community plugins within Obsidian's settings, but plugins are required to use this vault to its full potential.

From here on out, you can read through the README file from within Obsidian itself. It should be the very last item in the file navigator to the left.

If you're COMPLETELY new to Obsidian, I recommend exploring the sandbox vault that Obsidian provides out of the box. Press `Command + P` and search for `Open sandbox vault`. It'll guide you through Obsidian based on your experience level.

## Give me a tour of the vault!

- **00 SYSTEM**
	- **attachments** - Where images will automatically be placed into when you insert an image into your notes
	- **templates** - Where your templates live
- **01 ATLAS** - Notes meant primarily for vault navigation, including but not limited to Bases and notes linking to multiple other notes
	- [[Home]] - Vault homepage
- **COURSES** - Course pages; every note is dedicated to one course and should be named after the course code (for example, BIO-203); each course page will have an overview of the lectures and textbook notes you've taken for the class
- **LECTURE NOTES** - Lecture notes
- **SEMESTERS** - Semester pages; every note is dedicated to one semester/term, and gives an overview of the courses you took during that term
- **SOURCE NOTES** - Source notes (such as textbook notes)
- Everything else can go in the root folder of your vault. You can also make a MISC folder if you don't want notes cluttering the file navigator.

When you first enter the vault, you'll be sent to **/01 ATLAS/Home**. This is your dashboard. It currently has bases that will automatically list the courses and semesters you add to your vault, but you can add whatever information you want to see when you first open up Obsidian. 

This Student Vault is designed to be as simple as possible, with little folder structure and relying heavily on Obsidian's linking, bases, and search functions. You can add more folders and structure if needed as you use this vault to reduce visual clutter in the file navigator, but using links and bases with properties will handle most of the organization for you.

## What are properties?

Properties can be assigned to notes so that they can be organized and listed out by bases. An example of a base can be found [[Templates.base|here]]. This base automatically lists all notes within the **templates** folder. 

There are also examples of bases on the [[Home]] page. These are bases that filter through all of your notes based on properties called **tags**. For example, everything tagged with *course* will show up in the Courses base. Everything tagged with *semester* will show up in the Semesters base. 

For the immediate use of this vault, knowledge about bases is not required, but if you want to customize and add more, there are videos on YouTube that cover the details extensively.

## What plugins are used in this vault?

- [Editing Toolbar](https://github.com/pkm-er/obsidian-editing-toolbar)
	- An editing toolbar similar to MS Word to help you with formatting, especially if you're unfamiliar with Markdown.
- [Excalidraw](https://github.com/zsviczian/obsidian-excalidraw-plugin)
	- For making visual notes/diagrams/charts/etc. You can remove or disable this one if you only plan on making text-based notes.
- [Git](https://github.com/vinzent03/obsidian-git)
	- For syncing your vault to GitHub. You can remove this one if you don't plan on backing up your vault there.
- [Homepage](https://github.com/mirnovov/obsidian-homepage)
	- Sets a specified note to automatically open upon entering your vault.
- [Latex Suite](https://github.com/artisticat1/obsidian-latex-suite)
	- Makes typing out mathematical equations using MathJax faster and easier. You can remove or disable this one if you don't plan on taking any notes for math classes in Obsidian, or if text expansions are getting in the way of typing.
- [Notebook Navigator](https://github.com/johansan/notebook-navigator)
	- Overhauls Obsidian's file navigator UI, enables you to pin files as Shortcuts, and much more. You can remove or disable this one if you want to use Obsidian's core file navigator.
- [Outliner](https://github.com/vslinko/obsidian-outliner)
	- Allows you to work with your lists as you would in Workflowy or RoamReasearch, using keyboard shortcuts or dragging and dropping list items to reorganize them.
- [QuickAdd](https://github.com/chhoumann/quickadd)
	- Allows you to quickly add certain types of notes based on specified templates, and within specified folders. Highly recommended to keep.
- [Style Settings](https://github.com/obsidian-community/obsidian-style-settings)
	- For customizing the themes you download.
- [Templater](https://github.com/silentvoid13/Templater)
	- A robust and versatile template plugin. Highly recommended to keep.
